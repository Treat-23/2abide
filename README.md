A crowd-reported, GPS-tagged monitoring system for road damage — logging every pothole from first report to fixed, so city crews fix what matters most, first.

Report a Pothole →
Every pothole starts as a report. A driver, cyclist, or pedestrian drops a pin, adds a photo, and the system logs GPS location, timestamp, and estimated severity.

Reports cluster on a live map, so repeated flags on the same stretch of road automatically escalate priority — no single complaint gets lost, and no ward gets left off the map.

Built for local roads, sized for a city: Faridabad-first, with a structure that scales to any municipality willing to plug in its ward boundaries .
